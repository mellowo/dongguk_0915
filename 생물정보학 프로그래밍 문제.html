파일 핸들 및 Fasta 포멧 파일 파싱

https://github.com/mellowo/dongguk_0915에 접속하여 파일을 다운 받는다.

  1. "This is a sequence file."이라는 내용을 가진 title.txt 파일을 만든다.

    $ python bioinfo_1.py
$ cat title.txt
This is a sequence file.
$

  2. 위에서 만든 title.txt 파일의 첫째줄을 읽어 다음과 같이 출력한다.

    $ python python bioinfo_2.py
The first line is: This is a sequence file.
$

  3. github 에서 받은 파일, _1.csv 파일의 내용중 site1에 해당하는 파일의 내용만 출력해 보자.

    $ python python bioinfo_3.py
Sample1 site 1: GA
Sample2 site 1: GT
Sample3 site 1: AA
Sample4 site 1: TA
$

  4. github 에서 받은 파일, _2.tsv 파일의 내용중 male의 평균 나이를 출력해 보자.

    $ python python bioinfo_4.py
male's average age: 24.2301
$

    $ python python bioinfo_7.py
The second line is: MPIGSKERPTFFEIFKTRCNKADLGPISLNWFEELSSEAPPYNSEPAEESEHKNNNYEPNLFKTPQRKPS
$

  8. sequence.protein.2.fasta의 첫째줄은 title이란 변수에, 둘째줄부터 끝까지는 개행문자를 없앤 후 seq이란 변수에 저장하고 다음과 같이 출력한다.

    $ python python bioinfo_8.py
title: >AAB07223.1 BRCA2 [Homo sapiens]
seq: MPIGSKERPT...QDTITTKKYI
$
